🏗️ Arquitectura de módulos NestJS
MóduloResponsabilidadEndpointsDependencias
RabbitModule Conexión AMQP singleton + reconexión auto Interno amqplib
MessagesModule Peek sin consumir permanentemente GET /queues/:q/messages RabbitModule
RequeueModule Requeue individual y batch concurrente POST /requeue{,/batch} RabbitModule

MetricsModule Management API via axios basic auth GET /metrics{,/overview} axios, ConfigService
QueuesModule Combina métricas + peek GET /queues{,:q/messages} Messages, Metrics
